Welcome to Flight Routes System

This system allows you to manage airport routes and perform various operations:

  • Add Route

    Add new routes between airports with position and duration information.

  • Search Node

    Find the Nth left or right node in an airport route.

  • Longest Route

    Find the longest node based on duration in the airport system.

  • Shortest Route

    Find the shortest node based on duration between two airports.
